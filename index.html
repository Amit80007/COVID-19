Fork Crest

What is Crest

Crest quickly tests any HTML web page for a simple way to solve your accessibility testing problems. Provide a website URL to our callable API and expose potential WCAG violations.

Currently supports:

  • Keyboard Focus Indicator
  • Closed Captioning/ Transcript Check
  • Heading Analysis

Getting Crest

Using Conda

  1. Download and Install Miniconda from conda.io
  2. For Windows, launch Miniconda prompt shell. For MacOS, launch the terminal.
  3. Create a new virtual environment and install packages. 
                      
                    conda create -n crest python pandas tqdm
                    conda activate crest
  4. If using cuda: 
                      
                    conda install pytorch cudatoolkit=10.1 -c pytorch
    else: 
                    
                  conda install pytorch cpuonly -c pytorch
  5. Clone the repository. More details can be found at docs.github.com
  6. Go to "x43_accessibility-" folder in your terminal/prompt shell. Run command. 
                      
                    pip install -r requirements.txt
  7. Install chromedriver. The details can be found in chromium.org.
  8. Download and Install google-chrome-stable from ubuntuupdates.org.
  9. Once all the required libraries are installed, run command in your terminal/prompt shell. 
                      
                    python server.py

Using Docker

  1. Clone the repository. More details can be found at docs.github.com.
  2. Download and Install docker from docker.com. Note: Make sure your docker server as well as crest image is running.
  3. Go to "x43_accessibility-" folder in your terminal/prompt shell. Run command. 
                      
                    docker build -t crest .
  4. Once the docker image is created, run it by executing the following command. 
                      
                    docker container run --name crest_container -p 3000:3000 crest
